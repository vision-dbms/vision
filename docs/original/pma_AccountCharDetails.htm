Vision Portfolio Management Application Layer: Application

| Headstart Applications | 

Application: AccountCharDetails
 

Summary:

The AccountCharDetails application is used to display fundamental data about each equity security held in an account.

Inputs:

Position Form Id Description 
1  id  Id of the account 
2  sort  Sorting criteria 

Analysis:

  • The id must reference an existing Account instance.
  • If parameter 2 or the sort field is supplied, it specifies the sorting criteria. By default, the report is sorted from highest percent of portfolio to lowest.
  • The basic report displays securities as rows and the following columns:
    • Security Id
    Security Name
    Percent of Portfolio
    Market Capitalization
    Price Earnings Ratio
    Price to Book Ratio
  • If no global currency has been specified, all monetary data is expressed in the account's base currency.

Key Data Feeds:

  • HoldingsFeed
  • FundamentalM

Related Reports:

  • AccountProfile
  • AccountHoldings
  • AccountCharacteristics

Sample Usage:

      Interface ApplicationWS runWith: "AccountCharDetails", "103" ;

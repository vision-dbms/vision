Application: AccountProfile

 

Summary:

The AccountProfile application is used to provide summary information about an account with links to more in depth information. The summary includes basic information about the account, a list of its largest holdings, and a summary of its holdings history.

Inputs:

Position Form Id Description 
1  id  Id of the account 

Analysis:

  • The id must reference an existing Account instance. If it does not, a directory of all accounts is displayed, with the option to sort the list by account ID or account name.
  • The display includes four sections:
    • 1. Basics - basic information about the account
    2. Top 5 Equity Holdings - security name, shares, market value, and percentage for the largest five equity holdings in the account
    3. Recent Holdings History - dates of the last five holdings updates and the market value and cash for each date
    4. Membership Information - for Portfolios, this section displays all the aggregates that include the portfolio ; for AggAccounts, this section displays the portfolios currently comprising the aggregate ; for IndexAccounts, this section displays the history membership definition rules; for CompositeAccounts, this section displays the accounts currently comprising the composite and their weights in the composite

Key Data Feeds:

  • HoldingsFeed
  • PortfolioAggregates
  • CompositeAccountMembers

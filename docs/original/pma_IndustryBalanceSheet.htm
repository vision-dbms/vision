Application: IndustryBalanceSheet

 

Summary:

The IndustryBalanceSheet application is used to display current balance sheet information for the largest companies in an industry based on total assets.

Inputs:

Position Form Id Description 
1  id  Id of the industry 

Analysis:

  • The id must reference an existing Industry instance.
  • The basic report displays the balance sheet line items as rows and the top 5 companies in the industry (by assets) as columns.

Key Data Feeds:

  • FundamentalA
  • CompanyToIndustry

Related Reports:

  • IndustryProfile
  • CompanyBalanceSheet
  • IndustryIncomeStatement

Sample Usage:

    Interface ApplicationWS runWith: "IndustryBalanceSheet", "530" ;

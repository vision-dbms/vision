Application: SecurityProfile

 

Summary:

The SecurityProfile application is used to provide summary information about a security with links to more in depth information. The summary includes basic information about the security, financial highlights, and portfolio highlights.

Inputs:

Position Form Id Description 
1  id  Id of the security 
2  -  Id of currency override 

Analysis:

  • The id must reference an existing Security instance.
  • If no global currency has been specified, all monetary data is expressed in the security's base currency.
  • The display includes four sections:
    • 1. Basics - basic information about the security including alternate ids
    2. Relationships - basic classifications of the security including company, country, industry, sector, and universe memberships
    3. Financial Highlights - latest fiscal dates and data
    4. Portfolio Highlights - list of portfolios holding the security, the number of shares held, and the total market value

Key Data Feeds:

  • CompanyMaster
  • SecurityMaster
  • FundamentalA
  • FundamentalQ
  • FundamentalM
  • PricesFeed
  • HoldingsFeed

Related Reports:

  • SecurityPricingReport
  • CompanyEarningReport
  • CompanyBalanceSheet
  • CompanyIncomeStatement
  • HoldingsForSecurity

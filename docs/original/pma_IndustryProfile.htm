Application: IndustryProfile

 

Summary:

The IndustryProfile application is used to provide summary information about an industry with links to more in depth information. The summary includes basic information about the industry, financial highlights, and a summary of its membership history.

Inputs:

Position Form Id Description 
1  id  Id of the industry 

Analysis:

  • The id must reference an existing Industry instance.
  • If no global currency has been specified, all financial data is converted to US dollars. Fiscal year data is accessed as of the year end on or before the evaluation date.
  • The display includes four sections:
    • 1. Basics - basic information about the industry
    2. Financial Highlights - summary financial statistics based on companies in the industry
    3. Membership History - number of members included in the industry on each date that the membership changed

Key Data Feeds:

  • IndustryMaster
  • FundamentalA
  • FundamentalM

Related Reports:

  • IndustryHighlights
  • IndustryMembership
  • IndustryBalanceSheet
  • IndustryIncomeStatement

Application: CompanyEarningsReport

 

Summary:

The CompanyEarningsReport application is used to display historical earnings information for a company.

Inputs:

Position Form Id Description 
1  id  Id of the company 

Analysis:

  • The id must reference an existing Company instance.
  • The basic report displays the last 24 months of:
    • Date
    Price
    Earnings per Share
    Earnings per Share (from operations)
    P/E Ratio
    Shares outstanding
    Market Capitalization

Key Data Feeds:

  • FundamentalM

Related Reports:

  • CompanyProfile
  • SecurityProfile

Sample Usage:

    Interface ApplicationWS runWith: "CompanyEarningsReport", "DELL" ;

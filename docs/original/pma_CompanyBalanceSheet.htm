Application: CompanyBalanceSheet

 

Summary:

The CompanyBalanceSheet application is used to display the last three years of balance sheet information for a company.

Inputs:

Position Form Id Description 
1  id  Id of the company 

Analysis:

  • The id must reference an existing Company instance.
  • The basic report displays the balance sheet line items as rows and the last three fiscal year end dates as columns.

Key Data Feeds:

  • FundamentalA

Related Reports:

  • CompanyBalanceSheetQ
  • CompanyIncomeStatement
  • CompanyProfile

Sample Usage:

    Interface ApplicationWS runWith: "CompanyBalanceSheet", "DELL" ;

Enter Security:
